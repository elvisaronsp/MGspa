# Quasar App

> A Quasar project

## Build Setup

``` bash
# install dependencies
$ npm install

# serve with hot reload at localhost:8080
$ quasar dev

# build for production with minification
$ quasar build

# lint code
$ quasar lint
```

## Se Quasar CLI for atualizado

``` bash
# Instalar Quasar CLI local
npm install --save-dev quasar-cli@0.6.5

# Instalar NPX Global
npm install -g npx

# Rodando o projeto
npx quasar dev
```
